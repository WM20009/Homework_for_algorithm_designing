# 最近点对可视化项目

## 项目描述
本项目实现了一个最近点对（Closest Pair）问题的可视化解决方案，用于算法设计与分析课程的作业1。

## 功能
1. **随机生成点集**：支持在二维平面上随机生成一组点。
2. **最近点对计算**：使用分治算法高效计算最近点对。
3. **可视化展示**：通过图形界面展示点集及最近点对的连线。

## 使用方法
1. 确保已安装 Python 3.x 环境。
2. 安装依赖库：
   ```bash
   pip install matplotlib
   ```
3. 运行脚本：
   ```bash
   python closest_pair_visualization.py
   ```

## 文件结构
- `closest_pair_visualization.py`：主程序文件，包含算法实现和可视化逻辑。

## 示例
运行程序后，程序会生成一个随机点集，并标记出最近的点对。
